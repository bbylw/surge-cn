
<h1 align="center"><code>surge</code> - 静态网站发布工具</h1>

### 介绍

Surge 在全球 10 个地区拥有超过 1400 万次部署，是发布静态网页内容的首选工具之一。Surge 慷慨的免费套餐支持无限自定义域名和无限部署次数，使其成为暂存环境和生产构建的宝贵工具。

![surge-help](https://github.com/user-attachments/assets/5b3e7e92-008e-48b6-9c61-dc4cbf1bf22c)

### 关于 Surge

Surge 基于现代化的 API 优先架构构建，从一开始就设计为可编程和可组合的。CLI 中的所有功能都可以通过 API 访问，便于集成到自动化工作流、CI/CD 管道和自定义工具中。

这种 API 优先的方法使 Surge 非常适合 AI 驱动的工作负载，在这些场景中，智能体和自动化系统需要大规模部署、更新和管理网站资产。无论你是在构建 AI 驱动的开发工具、自动化内容管道，还是基于智能体的网页发布系统，Surge 都能提供支持大规模程序化部署的基础设施。

### 安装

    npm install -g surge

### 发布

    surge . hello.surge.sh
